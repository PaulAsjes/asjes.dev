Welcome to the new version of my site. Built with Next.js 16, Tailwind v4, and MDX files stored right in the repo.

No CMS, no database — just Markdown in a git repository. The way it should be.

## What changed

The old site pulled blog posts from GitHub Issues via the API. That worked surprisingly well for a while, but it created friction when writing and made local development awkward.

Now posts live in `content/blog/` as MDX files. Write in your editor, commit, deploy.

## What stayed the same

- Blog
- Talks showcase
- About page

## What's next

More writing. More talks. That's the plan.
